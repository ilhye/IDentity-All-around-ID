# IDentity-All-around-ID
IDentity-All-around-ID is a web-based application designed to simplify government ID application and identify government-issued IDs based on their age. It provides features like input automation, ID information, and accessible interface.

## 🛠️ Features
- Form Automation: Quickly fill out forms using pre-stored identity data.
- Secure Backend: Store sensitive data securely with Firebase.
- Secure Source code: The HTML file for this system is encrypted so that only authorize people will only have the access.
- Accessible Interface: A user-friendly, inclusive design compliant with accessibility standards (WCAG).
- Responsive Design: Optimized for use across devices, including desktops, tablets, and smartphones.

## 📦 Get Started
### Prerequisites
Before you begin, ensure you have the following installed on your system:
- Python 3.7+
- Google Chrome (latest version)
- ChromeDriver (must be compatible with Chrome version)
